敘述統計量

Summary functions take vectors as input and return one value

The name will be the name of the variable in the result. The value should be an expression that returns a single value like min(x), n(), or sum(is.na(y)).

flights %>% summarise(avg_dep_delay = mean(dep_delay, na.rm = T), arr_delay_sd = sd(arr_delay, na.rm = T))
## # A tibble: 1 x 2
##   avg_dep_delay arr_delay_sd
##           <dbl>        <dbl>
## 1          12.6         44.6
#為什麼na.rm沒有去掉會變成NA?
flights %>% summarise(IQR_dep_delay = IQR(dep_delay, na.rm = T), max_dep_delay = max(dep_delay, na.rm = T))
## # A tibble: 1 x 2
##   IQR_dep_delay max_dep_delay
##           <dbl>         <dbl>
## 1          16.0          1301
